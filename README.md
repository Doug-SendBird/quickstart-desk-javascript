# SendBird Desk SDK for JavaScript

This project provides SendBird Desk Widget sample.

## Install

- npm install
- npm run build

## Demo

- npm run demo
- (in browser) http://127.0.0.1:8888

## Browser support

- Modern browsers supporting ES6+ (Chrome, FireFox, Edge, Safari, etc)
- IE 11+
- Mobile browsers (Android/iOS)
